# User Guide

## Projects
1. Open **Projects**.
2. Enter project name.
3. Pick a color using the color picker.
4. Save and use the project in note creation.

## Tags
1. Open **Tags**.
2. Create a tag name and optional color.
3. In note creation, select one or more tags.
4. Use tag filter on Notes page to narrow results.

## Summarize and Translate
1. Open a note detail page.
2. Click **Summarize** or **Translate**.
3. Wait while processing state is active.
4. View summary and translation results in the detail view.

## Share Notes
1. Open note detail page.
2. Open the share form.
3. Choose channel: Email, WhatsApp, or Notion.
4. Choose content: summary, translation, both, or full transcription.
5. Provide destination and optional target language.
6. Submit and review success/error feedback.
